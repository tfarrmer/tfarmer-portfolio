# TFarmer Portfolio

This repository holds my personal portfolio website, built to showcase who I am, the work I do, and the journey that got me here.

## About the Website

- The site is a portfolio and personal story hub.
- It documents my journey, skills, projects, and the direction I am moving in.
- It is designed to present my experience and creative progress in a simple, polished way.

## Visit the Website

You can view the live site here:

https://tfarrmer.github.io/tfarmer-portfolio/

## Structure

The site is built using plain HTML and CSS, with content organized across a few pages:

- `index.html` — the homepage and portfolio overview
- `about.html` — details about my background and journey
- `skills.html` — a summary of the skills I bring to projects
- `contact.html` — ways to get in touch
- `styles.css` — the shared styling for the site

## Purpose

This portfolio is not just a collection of work; it is a record of growth. It reflects where I started, what I have learned, and where I am headed next.

## Notes

- The project is ready to publish on GitHub Pages.
